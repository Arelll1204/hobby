Halaman 3 — Inline Styling Lengkap

font-family

font-family: Arial, sans-serif — ini font bawaan sistem.

font-family: font1.ttf — custom font 1 [plus jakarta sans] (fallback Arial).

font-family: font2.ttf — custom font 2 [product sans] (fallback Georgia).

font-family: font3.ttf — custom font 3 [black ops one] (fallback Courier New).

font-size

font-size: 10px — kecil banget

font-size: 14px — ukuran badan teks umum

font-size: 18px — agak besar

font-size: 24px — besar

font-size: 1rem — relatif terhadap root (biasanya = 16px)

font-size: 1.5em — 1.5x dari parent

color

color: #1E2761 — navy (hex)

color: #C44545 — merah (hex)

color: #5EA854 — hijau (hex)

color: rgb(242, 201, 76) — kuning (rgb)

color: steelblue — nama warna CSS

background-color

background-color: #CADCFC — biru muda

background-color: #F2C94C — kuning

background-color: #1E2761; color: #FFFFFF — navy + teks putih

background-color: hijau + teks putih

padding & margin

padding: 20px — ruang dalam elemen (semua sisi sama)

padding: 8px 24px — atas-bawah 8px, kiri-kanan 24px

margin: 40px 0 — jarak luar atas-bawah 40px, kiri-kanan 0

border & border-radius

border: 2px solid #1E2761 — border navy solid

border: 3px dashed #C44545 — border merah putus-putus

border-radius: 8px — sudut membulat

border-radius: 50px — pill shape

font-weight & font-style

font-weight: normal — teks biasa

font-weight: bold — tebal

font-weight: 300 — tipis (light)

font-weight: 900 — sangat tebal (black)

font-style: normal

font-style: italic — miring

Contoh Gabungan

Semua properti digabung dalam satu style attribute.

Kombinasi lain: Georgia italic, kuning, border navy.
